# Window Functions

## هدف

Window Function ها یکی از مهم‌ترین قابلیت‌های SQL Server برای تحلیل داده هستند و در گزارش‌های حرفه‌ای کاربرد فراوانی دارند.

## مفاهیم استفاده‌شده

* RANK()
* LAG()
* LEAD()
* OVER()

## گزارش‌های موجود

### Customer Ranking

رتبه‌بندی مشتریان بر اساس میزان خرید.

### Running Totals

محاسبه فروش تجمعی.

### Month-over-Month Analysis

مقایسه فروش با دوره قبلی.

Window Function ها در تحلیل داده، داشبوردهای مدیریتی و مصاحبه‌های فنی SQL Server اهمیت بالایی دارند.
