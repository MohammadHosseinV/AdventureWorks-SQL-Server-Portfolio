# تحلیل فروش

## مسئله کسب‌وکار

مدیریت سازمان نیاز دارد روند درآمد و عملکرد فروش را در طول زمان بررسی کند.

## هدف

تحلیل داده‌های فروش به منظور شناسایی روند رشد، الگوهای فروش و میانگین ارزش سفارشات.
