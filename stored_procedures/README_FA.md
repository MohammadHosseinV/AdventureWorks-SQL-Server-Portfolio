# Stored Procedures

## هدف

Stored Procedure ها برای استانداردسازی منطق گزارش‌گیری و استفاده مجدد از کد در SQL Server استفاده می‌شوند.

## مفاهیم استفاده‌شده

* CREATE PROCEDURE
* Parameters
* Reusable SQL Logic

## Procedure های پیاده‌سازی‌شده

### usp_get_customer_orders

نمایش سفارشات یک مشتری مشخص.

### usp_product_sales_report

گزارش فروش محصولات.

## فایل‌های موجود

* usp_get_customer_orders.sql
* usp_product_sales_report.sql

## مزایا

* استفاده مجدد از کد
* افزایش امنیت
* ساده‌سازی گزارش‌گیری
* نگهداری آسان‌تر
