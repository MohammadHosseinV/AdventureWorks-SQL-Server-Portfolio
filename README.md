# AdventureWorks SQL Portfolio

A portfolio project built on Microsoft's AdventureWorks database to demonstrate SQL Server skills through business-oriented data analysis and reporting.

## Skills Demonstrated

- SQL Query Writing
- INNER JOIN / LEFT JOIN
- GROUP BY & HAVING
- Aggregate Functions
- Subqueries
- Common Table Expressions (CTE)
- Views
- Window Functions
- Indexing Basics
- Query Performance Analysis

## Project Structure

### Sales Analysis
Analysis of yearly and monthly sales performance.

### Customer Analysis
Customer purchasing behavior and top customer identification.

### Product Analysis
Best-selling products, product performance, and unsold products.

### Revenue Analysis
Revenue breakdown by product categories.

### Window Functions
Ranking, running totals, and trend comparison.

### Indexing
Basic indexing and query optimization examples.
