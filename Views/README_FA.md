# View ها

## هدف

در بسیاری از سیستم‌های سازمانی، گزارش‌ها به صورت مستقیم از جداول اصلی تولید نمی‌شوند. به همین دلیل از View ها برای ساده‌سازی کوئری‌ها، افزایش خوانایی و ایجاد لایه گزارش‌گیری استفاده می‌شود.

## مفاهیم استفاده‌شده

* CREATE VIEW
* Aggregate Functions
* JOIN
* GROUP BY

## View های پیاده‌سازی‌شده

### vw_top_products

نمایش محصولات پرفروش به همراه تعداد فروش.

### vw_customer_sales

نمایش مجموع فروش هر مشتری.

### vw_category_revenue

نمایش درآمد هر دسته‌بندی محصول.

## فایل‌های موجود

* vw_top_products.sql
* vw_customer_sales.sql
* vw_category_revenue.sql

## کاربرد

این View ها می‌توانند به عنوان منبع گزارش‌گیری برای داشبوردها، گزارش‌های مدیریتی و ابزارهای BI مورد استفاده قرار گیرند.
